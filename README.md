# Character Creator

A beginner-friendly Python project that lets players choose between three different character classes and view their stats, abilities, and descriptions.

## About

This project was created as part of my Python learning journey. Players can select from three classes:

* Warrior
* Mage
* Archer

Each class has its own unique stats, abilities, and playstyle.

The goal of this project is not only to practice Python fundamentals but also to serve as a foundation for a larger RPG-style character system as I continue learning new programming concepts.

## Features

* User class selection
* Unique class descriptions
* Character stats display
* Special abilities for each class
* Simple text-based interface
